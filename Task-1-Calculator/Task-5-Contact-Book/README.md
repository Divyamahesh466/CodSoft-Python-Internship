# Contact Book 📒

## Description
A Python-based Contact Book application used to store and manage contact details efficiently. Users can add, search, update, and delete contacts easily.

## Features
- Add new contacts
- Search saved contacts
- Update contact details
- Delete contacts
- Display all contacts
- Simple and user-friendly interface

## Technologies Used
- Python
