# Password Generator 🔐

## Description
A simple Python-based Password Generator application that creates strong and secure passwords using letters, numbers, and special characters.

## Features
- Generate random passwords
- User-defined password length
- Uses alphabets, numbers, and symbols
- Simple and user-friendly interface
- Improves password security

## Technologies Used
- Python
